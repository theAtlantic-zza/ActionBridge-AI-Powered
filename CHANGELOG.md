## Changelog

### v1.0.0 — 2026-04-23

**ActionBridge: execution closure MVP (shippable)**

- **Execution closure workflow**: input → analyze → review/edit → export
- **Evidence-first UI**: each extracted item includes a source excerpt for verification
- **Human-in-the-loop**: confidence + needs review via explicit confirmation checkbox
- **Export**: copyable checklist + download as Markdown and CSV
- **BYOK**: connect any OpenAI-compatible API (key stored locally in browser)
- **Demo-ready without keys**: mock mode with realistic sample flows (CN/EN)
- **Document input**: import `.txt`, `.md`, `.pdf`
- **OCR fallback for scanned PDFs**: optional OCR path when PDF has no text layer (with cost/privacy hint)
